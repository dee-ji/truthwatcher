# Truthwatcher UI

React + TypeScript + Vite UI foundation for Truthwatcher.

## Run locally
1. Install dependencies:
   ```bash
   npm install
   ```
2. Set API endpoint if needed:
   ```bash
   cp .env.example .env
   ```
3. Start dev server:
   ```bash
   npm run dev
   ```

## Build
```bash
npm run build
```
